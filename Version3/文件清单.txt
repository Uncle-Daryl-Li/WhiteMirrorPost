═══════════════════════════════════════════════════════════════
  文件用途详解
═══════════════════════════════════════════════════════════════

【前端文件】

  Landing page.html (20KB)
  ├─ 作用：项目首页和登录界面
  ├─ 特性：极光动画、用户登录
  └─ 跳转：点击按钮进入 SaaS.html

  SaaS.html (40KB)
  ├─ 作用：主工作区界面
  ├─ 特性：输入框、侧边栏设置、结果展示
  ├─ API调用：发送请求到 http://localhost:8000/api/generate
  └─ 动画效果：极光、霓虹光效

【后端文件】

  backend.py (6KB)
  ├─ 作用：核心生成逻辑
  ├─ 函数：generate_posters() - 调用Gemini API
  ├─ 输入：提示词、参数
  └─ 输出：PIL Image对象列表

  server.py (4KB)
  ├─ 作用：FastAPI Web服务器
  ├─ 端点：GET /  (健康检查)
  │        POST /api/generate  (生成海报)
  ├─ CORS：允许前端跨域访问
  └─ 端口：8000

【配置文件】

  .env (54字节)
  ├─ 作用：环境变量配置
  ├─ 必填：GOOGLE_API_KEY=你的实际Key
  └─ 注意：不要上传到公开仓库

  requirements.txt (135字节)
  ├─ 作用：Python依赖列表
  ├─ 内容：fastapi, uvicorn, pydantic,
  │        python-dotenv, google-genai, Pillow
  └─ 安装：pip install -r requirements.txt

【启动脚本】

  一键启动.bat (2.8KB)
  ├─ 作用：全自动启动
  ├─ 步骤：检查环境 → 启动后端 → 启动前端 → 打开浏览器
  └─ 推荐：新手使用

  start_backend.bat (287字节)
  ├─ 作用：启动FastAPI服务器
  ├─ 命令：python server.py
  └─ 端口：8000

  start_frontend.bat (307字节)
  ├─ 作用：启动HTTP服务器
  ├─ 命令：python -m http.server 8080
  └─ 端口：8080

【文档文件】

  开始使用.txt
  └─ 快速上手，3步启动

  快速启动.txt
  └─ 单页速查，常见问题

  README.md
  └─ 完整教程，详细说明

  项目说明.md
  └─ 项目结构，功能特性

  frontend-backend_problem.md (17KB)
  └─ 技术细节，问题解决

【工具文件】

  test_connection.py (2.3KB)
  ├─ 作用：测试前后端连接
  ├─ 功能：健康检查、API测试
  └─ 运行：python test_connection.py
