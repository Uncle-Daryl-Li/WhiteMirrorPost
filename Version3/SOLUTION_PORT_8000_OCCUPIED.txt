Step 1: Start servers on different port
  ► Double click: start_with_port_8001.bat

Step 2: Update SaaS.html to use port 8001

  Open SaaS.html with a text editor
  Find this line (around line 464):
    const response = await fetch('http://localhost:8000/api/generate', {

  Save the file.

Step 3: Restart the frontend
  Close the Frontend window
  Run: start_with_port_8001.bat again
